> Ajnix suit le [semantic versioning](https://semver.org/lang/fr/). Les changements majeurs sont annoncés avec au moins 30 jours de préavis.

## [0.1.0] · 2026-04-XX

Première release publique.

### Ajouté

**Tracking**
- Tracker JavaScript léger (6,8 Ko gzippé)
- Capture des UTMs et 6 click IDs (Google, Meta, TikTok, Microsoft, LinkedIn, X)
- Détection automatique des touchpoints et sessions
- Détection de bot intégrée (canvas fingerprinting exclu)

**Attribution**
- Moteur d'attribution last-touch avec fenêtre 30 jours
- Table `attribution_results` future-proof pour les modèles Pro

**WooCommerce**
- Tracking automatique des purchases et refunds
- Décomposition du revenu : gross, net, COGS, fees, refunded, gross profit
- Edition inline du COGS par produit avec calcul de marge
- Détection des frais de transaction pour Stripe, PayPal, Square

**Objectifs personnalisés**
- Builder visuel pour 6 types : URL, click, form submit, custom event, scroll, time
- Jusqu'à 5 objectifs actifs en free (filter `ajnix_goals_free_limit` pour ajuster)

**Intégrations formulaires (auto-détectées)**
- Contact Form 7
- WPForms
- Gravity Forms
- Fluent Forms

**Coûts et ROI**
- Coûts manuels avec prorata sur périodes chevauchantes (jusqu'à 10 en free)
- Product COGS avec calcul de marge
- Frais de transaction agrégés
- Summary ROI par canal

**UTM Link Builder**
- Création de liens UTM avec preview live
- Autocomplete basé sur l'historique
- Export SVG QR code
- Compteur de conversions par lien

**Intégrations consentement**
- Complianz
- CookieYes
- Cookiebot
- Iubenda
- Custom event hook

**API développeur**
- Fonctions PHP publiques (13 fonctions documentées)
- Hooks (13 actions)
- Filters (20 filters)
- REST API complète sous `/wp-json/ajnix/v1/`
- JS API publique via `window.ajnix()` et `window.AjnixTracker`

**Privacy**
- IP tronquée avant stockage (IPv4 /24, IPv6 /48)
- Aucune PII stockée
- Respect de Do Not Track
- Self-hosting du tracker JS (un toggle)

**Opérations**
- Daily aggregation via wp-cron
- Retention purge hebdomadaire (events 90j, sessions 180j)
- Multisite : chaque site a son `site_id` indépendant

### Configuration requise

- WordPress 5.8+
- PHP 7.4+
- MySQL 5.7+ / MariaDB 10.3+ avec InnoDB

### Documentation

- Documentation complète publiée sur [ajnix.com/docs](https://ajnix.com/docs)
- Developer reference dans `docs/developers.md`
- Security audit publié

## Politique de versionnage

- **MAJOR** (1.0 → 2.0) : breaking changes sur l'API publique (rare, préavis 30j)
- **MINOR** (1.0 → 1.1) : nouvelles features, compatibilité préservée
- **PATCH** (1.0.0 → 1.0.1) : bug fixes uniquement

Les fonctions, hooks et filters documentés dans [Developer Reference](/fr/docs/reference/php-api) sont **stables** au sein d'une version majeure.

Tout ce qui est préfixé `_ajnix_internal_*` ou marqué "Phase 2" peut changer entre versions mineures.
